🎬 Chatbot Netflix Style – Sistema Web

Projeto de chatbot interativo inspirado na interface da Netflix, desenvolvido como parte de um projeto acadêmico na área de Engenharia de Software.

O sistema simula um assistente virtual para plataforma de vídeos, com interface moderna, tema escuro e experiência semelhante a sistemas de streaming.

🚀 Demonstração

O chatbot funciona como um widget flutuante, permitindo que o usuário abra o chat e interaja com o assistente dentro da própria página.

Principais elementos da interface:

Botão flutuante de chat

Interface estilo Netflix Dark UI

Área de mensagens dinâmica

Campo de envio de perguntas

Estrutura pronta para integração com IA

🧠 Funcionalidades

✔ Chatbot flutuante na tela
✔ Interface inspirada em plataformas de streaming
✔ Abertura e fechamento do chat
✔ Campo de envio de mensagens
✔ Área de respostas do assistente
✔ Layout responsivo
✔ Estrutura modular (HTML + CSS + JS)
